# Supervised Models Lab

## Overview

This repository contains machine learning exercises focused on supervised learning models. The lab involves implementing and analyzing various supervised learning algorithms including classification and regression techniques.

## Setup Instructions

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Setting Up the Virtual Environment

To replicate this environment on another device, follow these steps:

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Supervised-Models-Lab
   ```

2. **Create a virtual environment**
   ```bash
   python -m venv venv
   ```

3. **Activate the virtual environment**
   
   On Windows:
   ```bash
   venv\Scripts\activate
   ```
   
   On macOS/Linux:
   ```bash
   source venv/bin/activate
   ```

4. **Install required packages**
   ```bash
   pip install -r requirements.txt
   ```

5. **Verify installation**
   ```bash
   pip list
   ```

### Deactivating the Environment

When you're done working, deactivate the virtual environment:
```bash
deactivate
```
